# [규칙]
### commit message
- 변경 : "mod filename (details)"
- 추가 : "add filename (details)"
- 삭제 : "del filename (details)"
- 긴급 : "fix filename (details)"

### syntax rule
- 변수 명	: 명사로 시작
- 함수 명	: 동사로 시작
- 매크로	: 용도별 접두어 통일
